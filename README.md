### 교육 데이터 분석 기반 컨설팅 프로젝트

학생 데이터를 활용하여 학습 성과를 분석하고, 학부모·교육기관의 의사결정을 지원하기 위한 데이터 기반 컨설팅 프로젝트입니다.

본 프로젝트는 단순한 데이터 분석에 그치지 않고, 교육 현장에서 발생하는 문제를 정의하고 데이터를 활용한 해결 방안을 제안하는 것을 목표로 수행되었습니다.

---

# 프로젝트 개요

중국에서 코딩 교육을 진행하면서 학생들의 학습 데이터를 직접 관리하고 분석한 경험을 바탕으로 진행한 프로젝트입니다.

학생들의 시험 성적, 출석률, 학습 흥미도, 수업 만족도 등의 데이터를 활용하여 학습 성과를 분석하고, 이를 학부모와 교육기관이 쉽게 이해하고 활용할 수 있도록 대시보드와 학습 리포트 형태로 시각화하였습니다.

또한 교육기관 운영 관점에서 학생 이탈 예측과 데이터 기반 운영 방안을 함께 제안하였습니다.

---

# 비즈니스 문제

교육 현장에서는 다음과 같은 문제가 존재합니다.

* 학생의 학습 성장을 객관적으로 측정하기 어렵다.
* 학부모에게 데이터 기반의 피드백을 제공하기 어렵다.
* 학생 이탈 가능성을 사전에 파악하기 어렵다.
* 교육기관 운영 현황을 한눈에 파악하기 어렵다.
* 데이터 기반 의사결정 체계가 부족하다.
